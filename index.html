Avatar

Avatars are user profile pictures, usually found in circular shapes. Avatars can be used for 'container' objects — projects, spaces, repositories, etcetera — to give them a recognisable visual identity.

Sizes Avatar

Sizes Avatar is available in 4 different sizes. You can use image in Avatar. You need to include class avatar and for size add class according to size xlg-avatar, lg-avatar, sm-avatar, xsm-avatar (e.g. class="avatar xlg-avatar")

Square Avatar

Square Avatar is available in 4 different sizes. You can use image in Avatar. You need to include class avatar and for size add class according to size sq-xlg-avatar, sq-lg-avatar, sq-sm-avatar, sq-xsm-avatar (e.g. class="avatar sq-xlg-avatar")

Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

Primary Alert

Alerts are available in 5 types- primary alert,secondary alert, success alert, danger alert, warning alert. To use this alert just use class name alert and also add class according to alert type- alert-primary, alert-secondary, alert-danger, alert-success, alert-warning. (e.g.class="alert alert-primary"). You can copy html part from below code snippet.
